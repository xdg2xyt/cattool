# 🐱 CATTOOL

Installer modular e elegante para Termux.

## ✨ Features

- OSINT tools
- Web Security
- Wireless tools
- Wordlists
- Interface colorida
- Sistema modular
- Logs automáticos
- Instalação automatizada

---

## 📦 Instalação

```bash
pkg update -y
pkg install git -y

git clone https://github.com/xdg2xyt/cattool
cd cattool

bash install.sh
```
